# MyUnityMainMenu

A clean and modern Unity Main Menu UI system featuring scalable navigation containers, custom audio settings dialogs, and dynamic screen switching without code conflicts.

##  Features

* 📱 **Dynamic Container Switching:** Smooth transitions between Main Menu, Options, and Audio sub-menus.
* 🛠️ **Inspector-Driven Logic:** Handled cleanly using custom logic via Unity's `On Click ()` events to prevent panel overlapping.
* 📐 **Scalable Layout:** Built using modern Unity UI Canvas components suited for responsive aspect ratios.
* 🔊 **Independent Popouts:** Specialized settings dialog containers (e.g., Audio Settings) designed to work independently from parallel menu panels.

---

##  Built With

* **Game Engine:** Unity (UI Toolkit / Canvas System)
* **Text Rendering:** TextMeshPro (TMP)
